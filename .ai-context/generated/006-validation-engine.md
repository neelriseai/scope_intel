## Debug Flow
1. Classify probable feature.
2. Query scope index.
3. Retrieve compact scope.
4. Query symbol context.
5. Open 2–5 source files.
6. Inspect related tests/logs.
7. Patch minimal set.
8. Run validation.
9. Refresh scope.

## Enhancement Flow
1. Identify feature boundary.
2. Query scope + dependent features.
3. Inspect entry points/tests.
4. Patch targeted classes.
5. Run impacted tests.
6. Update summaries.

## Risk Analysis
- Top change risk areas.
- Impacted files.
- Downstream dependencies.
